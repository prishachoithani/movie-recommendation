# Project Statement – Movie Recommendation System

##  Problem Statement
Users often struggle to decide which movie to watch next. This project aims to solve that problem by building a simple movie recommendation system that suggests movies based on user preferences and ratings.

##  Scope of the Project
- Works fully in the terminal/console  
- Allows interaction through a menu-driven interface  
- Implements search, rating, and recommendation features  
- Uses modular design with multiple Python files  
- Suitable for beginners learning Python and modular programming  

##  Target Users
- Students learning Python  
- Beginners exploring recommendation logic  
- Anyone wanting a simple console-based movie explorer  
- Developers experimenting with modular coding structures  

## High-level Features
- Movie Database:Stores movies and their genres  
- Search Engine: Find movies by name  
- Rating System: Users rate movies (1–5)  
- Recommendation Engine: Suggests top-rated movies within the same genre  
- Modular Architecture: Code divided into separate files for clarity
