# Movie Recommendation System (Python)

## Project Overview
This project is a modular Python-based Movie Recommendation System that allows users to explore movies, search by name, rate films, and receive smart recommendations based on genre and average ratings. It uses simple rule-based logic to suggest the best match according to user input.

##  Features
- View all available movies with genres  
- Search movies by partial or full name  
- Rate any movie from 1 to 5  
- Get genre-based recommendations  
- Modular structure with multiple Python files  
- Beginner-friendly and easy to expand  

## Technologies / Tools Used
- **Python 3**
- Standard Python modules (no external libraries)
- Modular programming
